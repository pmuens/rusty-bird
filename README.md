# Rusty Bird

A [Flappy Bird](https://en.wikipedia.org/wiki/Flappy_Bird) clone written in Rust following the ["Hands-on Rust" Book](https://pragprog.com/titles/hwrust/hands-on-rust) by Herbert Wolverson.

## Useful Commands

```sh
nix-shell

cargo new <name>

cargo build
cargo clean
cargo test

cargo run

cargo fmt

cargo clippy

cargo check
```
